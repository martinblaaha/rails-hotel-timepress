# README

This is a web application for managing the hotel's reservations, guests and rooms. For each Guests / Rooms / Reservations you can create new records, and edit or delete existing ones. You can run it on localhost and then connect to http://localhost:3000.

* Ruby version 4.0.1, Rails version 8.1.2

* Database may be loaded from the seeds.rb file by calling "rake db:seed"
